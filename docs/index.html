JAWT

Just A Weird Web Tool

Build minimal web applications using a declarative approach with Jawt Markup Language.

Declarative Web Development

JAWT enables you to build web applications using JML (Jawt Markup Language), a domain-specific language that makes creating modern JAWT apps intuitive and efficient.

Declarative Component Based Hot Reload Minimal Bundle
app/index.jml
// Create a new JAWT project
_doctype page index
import component Layout from "components/layout"
Page {
title: "Welcome to My App"
description: "Built with JAWT"
Layout {
content: "Hello, There!"
}
}
